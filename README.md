# free-hub POC

This project is a content-driven traffic and monetisation platform focused on competitions, offers, and digital engagement.

## Goals

- Generate low-cost traffic via display and DSP
- Engage users with content (competitions, offers, media)
- Monetise via native ads and referral flows
- Build a scalable traffic asset independent of Google

## POC Scope

This is a **static Proof of Concept (POC)**:
- No backend required
- Data served via JSON
- Fully deployable on GitHub Pages (free)

## Stack (POC)

- HTML / CSS / JavaScript (Vanilla or lightweight framework)
- GitHub Pages (hosting)
- JSON (data layer)

## Future Stack (Phase 2)

- Next.js (frontend upgrade)
- Vercel (hosting)
- Supabase (database)
- Node.js ingestion service

## Features (POC)

- Competition listing (cards)
- Search and filtering
- Category browsing
- Mobile-first UI

## Data Structure

Competitions are stored in `/data/competitions.json`

## Deployment

Deployed via GitHub Pages using GitHub Actions.
